# Home Task & Routine Planner

## Overview

The Home Task & Routine Planner is a Java-based application designed to help users organise recurring chores, habits, and personal tasks. The system allows users to input task preferences such as priority, effort, and timing, and generates a suggested weekly schedule that can be reviewed and adjusted.

## Features

- Create, edit, and delete recurring tasks
- Assign priority, effort, and preferred time
- Generate a suggested weekly schedule
- Review and adjust schedule placements
- Highlight tasks that may require manual review
- Offline task management with persistent local storage
- AI-assisted scheduling to suggest weekly plans

## Tech Stack

- Java (Object-Oriented Programming)
- Git & GitHub for version control

## Project Structure

- `main` – stable code
- `dev` – development branch
- `feature/*` – feature-specific branches
